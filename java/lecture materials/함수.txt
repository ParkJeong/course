함수



함수
하나의 기능을 수행하는 일련의 코드
어떤 기능을 수행하도록 미리 구현해 놓고 필요할 때마다 호출하여 사용할 수 있다.

매개변수
함수의 입력으로 받는 변수

함수가 하는 일을 코드로 구현하는 것

함수 이름
함수 기능과 관련 있게 만들어야한다.

void
반환할 값이 없다는 뜻의 예약어

함수호출
함수를 사용한다는 의미

함수를 호출하면 그 함수만을 위한 메모리 공간이 할당되는데, 이 메모리 공간을 스택이라고 부른다.

프로그램을 시작할 때 main()함수부터 호출하기 때문에 가장 먼저 main()함수와 함수에 포함된 변수를 저장할 메모리 공간이 생성된다

함수 내부에서만 사용하는 변수를 지역 변수라고 한다.
지역변수는 스택 메모리에 생성된다.

함수의 장점
함수를 사용하면 기능을 나누어 코드를 효율적으로 구현할 수 있다.
프로그램의 가독성이 좋아진다.
기능별로 함수를 구현해 놓으면 같은 기능을 매번 코드로 만들지 않고 그 기능의 함수를 호출하면 되니가 편리하다.
프로그램을 실행할 때 생긴 오류를 수정하는 디버깅 작업을 할 때도 편리하다.

하나의 함수에는 함수의 이름에 맞는 하나의 기능만을 구현하는 것이 중요하다.
